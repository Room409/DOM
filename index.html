getElementByID

Selecciona un elemento del DOM por su ID

document.getElementById("mi-id"); // Selecciona el elemento con el ID "mi-id"
getElementsByTagName

Selecciona todos los elementos del DOM que tienen una etiqueta específica.

document.getElementsByTagName("p"); // Selecciona todos los elementos

querySelector

Se puedes utilizar cualquier selector CSS válido como argumento

document.querySelector("p"); // Selecciona el primer elemento

querySelectorAll

Selecciona todos los elementos del DOM que coinciden con un selector CSS específico.

document.querySelectorAll("div p"); // Selecciona todos los párrafos que están dentro de elementos div
Cual es la diferencia?

QuerySelector y querySelectorAll permiten seleccionar elementos utilizando selectores CSS más flexibles, mientras que getElementById y getElementsByTagName seleccionan elementos específicos por su ID o etiqueta, respectivamente. Además, querySelector y getElementById devuelven un solo elemento, mientras que querySelectorAll y getElementsByTagName devuelven una lista de elementos.

Como acceder al DOM?

Para acceder al DOM (Document Object Model) utilizando JavaScript, puedes utilizar el objeto global document. El objeto document representa el documento HTML cargado en el navegador y proporciona métodos y propiedades para interactuar con él.

Ejemplos, getElementByID, getElementsByTagName, getElementsByClassName,document.querySelector, querySelectorAll
innerHTML vs textContent

InnerHTML se utiliza para acceder y manipular el contenido HTML de un elemento, mientras que textContent se utiliza para acceder y manipular el contenido de texto sin formato dentro de un elemento, sin interpretar las etiquetas HTML.

La propiedad innerHTML devuelve o establece el contenido HTML dentro de un elemento, incluyendo las etiquetas HTML. Permite obtener o modificar el contenido en formato HTML.
La propiedad textContent devuelve o establece el contenido de texto sin formato dentro de un elemento, sin interpretar las etiquetas HTML. Devuelve el contenido de texto plano dentro del elemento y elimina cualquier formato o etiquetas.
Proiedad "style"

La propiedad style en JavaScript se utiliza para acceder y manipular los estilos CSS de un elemento en el DOM. Proporciona acceso directo a las reglas de estilo aplicadas al elemento y permite realizar cambios en ellas.

let elemento = document.getElementById("mi-elemento").style = "color: red"
setAttritube y removeAttribute

Son dos métodos en JavaScript que se utilizan para establecer y eliminar atributos de elementos HTML en el DOM

setAttribute se utiliza para establecer un atributo en un elemento HTML específico.

removeAttribute se utiliza para eliminar un atributo de un elemento HTML específico.

document.getElementById("mi-elemento").setAttribute("class", "container");
document.getElementById("mi-elemento").removeAttribute("class");
firstChild - lastChild y nextSibling

Las siguientes propiedades se refieren a la navegación entre los nodos hijos y hermanos de un elemento:

firstChild: devuelve el primer nodo hijo de un elemento. Esto incluye cualquier tipo de nodo, como elementos HTML, texto o comentarios

lastChild: devuelve el último nodo hijo de un elemento. Al igual que firstChild, puede ser cualquier tipo de nodo.

nextSibling: devuelve el siguiente nodo hermano del elemento actual.

let primerHijo = document.getElementById("mi-elemento-padre").firstChild; console.log(primerHijo);
let siguienteHermano = document.getElementById("mi-elemento").nextSibling; console.log(siguienteHermano);
let ultimoHijo = document.getElementById("mi-elemento-padre").lastChild; console.log(ultimoHijo);
