getElementByID

Selecciona un elemento del DOM por su ID

document.getElementById("mi-id"); // Selecciona el elemento con el ID "mi-id"
getElementsByTagName

Selecciona todos los elementos del DOM que tienen una etiqueta específica.

document.getElementsByTagName("p"); // Selecciona todos los elementos

querySelector

Se puedes utilizar cualquier selector CSS válido como argumento

document.querySelector("p"); // Selecciona el primer elemento

querySelectorAll

Selecciona todos los elementos del DOM que coinciden con un selector CSS específico.

document.querySelectorAll("div p"); // Selecciona todos los párrafos que están dentro de elementos div
Cual es la diferencia?

QuerySelector y querySelectorAll permiten seleccionar elementos utilizando selectores CSS más flexibles, mientras que getElementById y getElementsByTagName seleccionan elementos específicos por su ID o etiqueta, respectivamente. Además, querySelector y getElementById devuelven un solo elemento, mientras que querySelectorAll y getElementsByTagName devuelven una lista de elementos.
